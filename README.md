# House Price Prediction with Linear Regression

## Project Overview

This project focuses on estimating house prices by applying a **Linear Regression** model. The objective is to study different housing characteristics and develop a predictive model that can estimate property prices based on important numerical and categorical features.

## Dataset

Housing.csv

The dataset includes details such as property area, number of bedrooms and bathrooms, parking facilities, furnishing status, and other variables that may affect housing prices.

## Objectives

* Examine and understand the housing dataset
* Perform data cleaning and preprocessing
* Transform categorical variables into numerical values
* Develop a Linear Regression prediction model
* Assess model performance using evaluation metrics
* Compare and visualize actual versus predicted house prices

## Steps Performed

1. Imported and explored the dataset
2. Checked and handled missing values
3. Converted categorical features using encoding techniques
4. Selected relevant features and divided the dataset into training and testing sets
5. Trained the model using the Linear Regression algorithm
6. Evaluated the model using Mean Squared Error (MSE) and R-squared (R²)
7. Created visualizations comparing predicted values with actual prices

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

## Results

The Linear Regression model successfully identified the relationship between various housing attributes and property prices. Evaluation metrics showed satisfactory prediction capability.

## Conclusion

This project offers practical experience in developing and assessing a regression-based machine learning model. It highlights how Linear Regression can be applied to solve real-world problems such as predicting housing prices.
